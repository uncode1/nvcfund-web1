Document Format Information

PDF Format

PDF (Portable Document Format) files maintain consistent formatting across all platforms and devices. This format is ideal for:

  • Official document submission
  • Printing with consistent layout
  • Sharing final versions
  • Document archiving

PDF documents require Adobe Acrobat Reader or similar software to view.

Microsoft Word Format

Microsoft Word (DOCX) files are fully editable. This format is ideal for:

  • Customizing agreement details
  • Adding specific clauses or terms
  • Making changes before finalization
  • Collaborative editing between parties

Word documents require Microsoft Office or compatible software to edit.
